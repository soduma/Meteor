### 🍀 Meteor는 아이폰의 기본앱 중 미리알림에서 영감을 받아 제작한 투두 앱입니다.
>대부분의 투두 앱이 가지고 있던 '하루 중 지정된 시각'에 '지정된 알림'을 받던 방식을 벗어나   
>**입력한 내용을 즉시 알림창에서 푸시받는** 그 자체로 하나의 메모장으로 사용할 수 있도록 구현하였습니다.

*한국어, 영어에 대해 현지화가 되어 있습니다.*
